# Career Compass

A single-page web app for the “Find Your Career” survey.

## Use it

Open `index.html` in a browser, or use your editor’s Live Server extension.

## Customize it

Select the ⚙ button in the upper-right corner to edit questions, career matches, scoring links, descriptions, and example roles. Changes are stored only in the current browser.

- Export settings as JSON to back them up or share them.
- Import JSON to restore a saved survey.
- Select “Save results” to open the print dialog and save a result as PDF.
